How to Add Art to the Website

    
    
      
  1. Fill out the form above and click "Generate My Listing."
    2. 
      
  2. Copy the code that appears in the green box.
    3. 
      
  3. Rename your photo to match the filename shown in the code (e.g. my-piece.jpg).
    4. 
      
  4. Go to your GitHub repository for this website.
    5. 
      
  5. For the photo: upload it to the images/kg/ folder (Kindness Grace) or images/wife/ folder (Market).
    6. 
      
  6. For the listing code: open art-data.json, find the right section ("wife" or "daughter"), and paste your code inside the [ ] brackets — add a comma after the last entry before pasting.
    7. 
      
  7. Save the file (commit changes) and the site will update automatically!
